# <a name="cannot-rename-with-different-drive"></a>ディスク名は変更できません
`Rename` ステートメントではファイル パスにドライブ名を指定できません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ファイル パスからドライブ文字を削除します。
