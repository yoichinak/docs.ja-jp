# <a name="argument-not-specified-for-parameter-parametername"></a>引数のパラメーターに指定されていない '\<parametername >'
必要なパラメーターの引数が指定されていません。  
  
 **エラー ID:** BC36569  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 指定されたパラメーターの引数を指定します。
