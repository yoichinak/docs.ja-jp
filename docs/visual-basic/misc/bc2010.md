# <a name="compilation-failed-message"></a>コンパイルに失敗しました:\<メッセージ >
コンパイルは、エラー メッセージで指定された環境が原因で失敗しました  
  
 **エラー ID:** BC2010  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- エラー メッセージを確認して、失敗の正確な原因を特定します。
