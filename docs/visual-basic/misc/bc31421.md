# <a name="member-is-already-declared-in-this-structure"></a>'\<メンバー >' は既にこの構造体で宣言されて
列挙体のメンバーがこの構造体内で 1 回以上宣言されています。  
  
 **エラー ID:** BC31421  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分な宣言を削除します。
