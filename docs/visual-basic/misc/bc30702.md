# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>インデックス '\<indexnumber >' 次元の'\<です >' が範囲外です
定義されている範囲を超えているインデックスがあります。  
  
 **エラー ID:** BC30702  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 配列の境界を越えないようにプロシージャのロジックを変更します。  
  
- 配列を再定義して値の範囲を広げます。
