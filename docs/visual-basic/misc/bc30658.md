# <a name="property-propertyname-with-no-parameters-cannot-be-found"></a>プロパティ '\<propertyname >' パラメーターが見つかりません
使用中の属性には、1 つまたは複数のパラメーターを必要とするプロパティがあります。  
  
 **エラー ID:** BC30658  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  プロパティのパラメーターを指定します。
