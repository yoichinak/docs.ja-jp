# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>'Continue'ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
'Continue' ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30780  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 'Continue' ステートメントは、デバッグ コードから削除します。
