# <a name="for-loop-not-initialized-visual-basic"></a>For ループが初期化されていません (Visual Basic)
`For...Next` ループの中央にジャンプしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ループへのジャンプを削除します。
