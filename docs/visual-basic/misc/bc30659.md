# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>プロパティまたはフィールド '\<名 >' は有効な属性型がありません
属性は、他の適切なコンス トラクターを持っている場合でも現在の有効な属性型を受け入れない属性コンス トラクターを使用する場合があります。  
  
 **エラー ID:** BC30659  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. この属性の使用法を確認します。
