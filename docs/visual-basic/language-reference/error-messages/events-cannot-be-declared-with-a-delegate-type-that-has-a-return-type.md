# <a name="events-cannot-be-declared-with-a-delegate-type-that-has-a-return-type"></a>イベントを、戻り値の型を持つデリゲート型で宣言することはできません。
関数プロシージャにデリゲートが指定されました。  
  
 **エラー ID:** BC31084  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `Sub` プロシージャにデリゲートを指定します。
