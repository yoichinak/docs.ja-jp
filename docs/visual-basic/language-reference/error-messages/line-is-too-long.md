# <a name="line-is-too-long"></a>行が長すぎます。
ソース テキストの行には、65535 文字を超えることはできません。  
  
 **エラー ID:** BC30494  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   65535 文字以下の行の長さを短くしてください。
