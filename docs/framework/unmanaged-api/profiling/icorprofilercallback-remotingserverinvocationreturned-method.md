# <a name="icorprofilercallbackremotingserverinvocationreturned-method"></a>ICorProfilerCallback::RemotingServerInvocationReturned メソッド
Notifies the profiler that the process has finished invoking a method in response to a remote method invocation request.  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT RemotingServerInvocationReturned();  
```
