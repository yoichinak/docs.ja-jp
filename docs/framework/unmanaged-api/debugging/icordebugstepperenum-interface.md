# <a name="icordebugstepperenum-interface"></a>ICorDebugStepperEnum インターフェイス
ICorDebugEnum のメソッドを実装し、ICorDebugStepper 配列を列挙します。  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[Next メソッド](../../../../docs/framework/unmanaged-api/debugging/icordebugstepperenum-next-method.md)|指定した数を取得`ICorDebugStepper`列挙体の現在位置からのインスタンス。|  
  
## <a name="remarks"></a>Remarks  
  
> [!NOTE]
>  このインターフェイスは、コンピューター間またはプロセス間でのリモート呼び出しをサポートしていません。
