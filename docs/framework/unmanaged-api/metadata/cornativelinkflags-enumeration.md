# <a name="cornativelinkflags-enumeration"></a>CorNativeLinkFlags 列挙型
ネイティブ コードをリンクするときに、リンカーが使用するフラグ値を提供します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
typedef enum  
{  
    nlfNone         = 0x00,  
    nlfLastError    = 0x01,  
    nlfNoMangle     = 0x02,  
    nlfMaxValue     = 0x03  
} CorNativeLinkFlags;  
```  
  
## <a name="members"></a>メンバー  
  
|メンバー|説明|  
|------------|-----------------|  
|`nlfNone`|Indicates no flags.|  
|`nlfLastError`|Indicates a `setLastError` keyword.|  
|`nlfNoMangle`|Indicates a `nomangle` keyword.|  
|`nlfMaxValue`|使用しません。|
