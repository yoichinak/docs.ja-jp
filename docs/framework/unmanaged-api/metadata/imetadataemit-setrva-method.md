# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA メソッド
指定したメソッドの相対仮想アドレスを設定します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,
    [in]  ULONG        ulRVA
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `md`  
 [in]ターゲット メソッドまたはメソッド実装のトークン。  
  
 `ulRVA`  
 [in]コードまたはデータ域のアドレス。
