# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog メソッド
Resets the edit-and-continue log and starts a new session.  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>パラメーター  
 `ResetENCLog` takes no parameters.
