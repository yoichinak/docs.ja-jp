# <a name="compiler-error-cs0228"></a>コンパイラ エラー CS0228
'type' に 'member' の定義がないか、アクセスできません  
  
 **System.Hashtable**、 **System.String**、または **System.Array**のシステム バージョンを置き換えた可能性があり、置き換えたものに `member`が含まれていません。  
  
 それ以外の場合は、Visual Studio を修復または再インストールする必要があります。
