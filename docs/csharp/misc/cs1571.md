# <a name="compiler-warning-level-2-cs1571"></a>コンパイラの警告 (レベル 2) CS1571
'construct' 上の XML コメントには 'parameter' の重複する param タグがあります  
  
 [/doc](../language-reference/compiler-options/doc-compiler-option.md) コンパイラ オプションを使用するときに、同じメソッド パラメーターに複数のコメントが検出されました。 重複する行の 1 つを削除します。  
  
 次の例では CS1571 が生成されます。  
  
```csharp  
// CS1571.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>An initial.</param>  
   /// <param name='Int1'>Used to indicate status.</param> // CS1571  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
