# <a name="compiler-error-cs0619"></a>コンパイラ エラー CS0619
'member' は使用されなくなりました: 'text'  
  
 クラスメンバーは、[Obsolete] (../programming-guide/concepts/attributes/common-attributes.md # Obsolete) 属性。クラスメンバーが参照されるときにエラーが発行されます。  
  
 次の例では CS0619 が生成されます。  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
