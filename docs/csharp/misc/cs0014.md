# <a name="compiler-error-cs0014"></a>コンパイラ エラー CS0014
必要なファイル 'file' が見つかりませんでした  
  
 コンパイラで必要なファイルがシステムにありません。 パスが正しいことを確認してください。 ファイルが Visual Studio のシステム ファイルの場合は、インストールを修復または削除して Visual Studio を完全に再インストールする必要があります。
