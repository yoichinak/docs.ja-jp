# <a name="compiler-error-cs1556"></a>コンパイラ エラー CS1556
Main メソッドに指定された 'construct' は有効なクラスか構造体でなければなりません。  
  
 [-Main](../language-reference/compiler-options/main-compiler-option.md)コンパイラオプションに、クラス名ではない識別子が渡されました。
