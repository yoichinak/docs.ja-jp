# <a name="compiler-error-cs2006"></a>コンパイラ エラー CS2006
コマンド ライン構文エラー:' Option ' オプションの ' text ' がありません  
  
 *option* の構文には追加のテキストが必要です。 詳しくは、「 [コンパイラ オプション](../language-reference/compiler-options/index.md)」をご覧ください。
