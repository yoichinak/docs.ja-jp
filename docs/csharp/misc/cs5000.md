# <a name="compiler-warning-level-1-cs5000"></a>コンパイラの警告 (レベル 1) CS5000
コンパイラ オプション '/option' が不明です  
  
 無効な [コンパイラ オプション](../language-reference/compiler-options/index.md) が指定されました。
