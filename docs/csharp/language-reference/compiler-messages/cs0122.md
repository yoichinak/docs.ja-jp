# <a name="compiler-error-cs0122"></a>コンパイラ エラー CS0122
'member' はアクセスできない保護レベルになっています  
  
 クラス メンバーの[アクセス修飾子](../../../csharp/language-reference/keywords/modifiers.md)で、メンバーのアクセスが禁止されています。 詳細については、「[アクセス修飾子](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md)」を参照してください。  
  
 (以下の例では示していませんが) この理由の 1 つは、フレンド アセンブリのターゲットで **/out** コンパイラ フラグを省略できることです。 詳細については、「[Friend Assemblies](../../../standard/assembly/friend-assemblies.md)」(フレンド アセンブリ) と「[/out (C# コンパイラ オプション)](../../../csharp/language-reference/compiler-options/out-compiler-option.md)」を参照してください。  
  
## <a name="example"></a>例  
 次の例では CS0122 が生成されます。  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
